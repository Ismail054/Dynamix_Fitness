# ForgeFit Gym Website

Responsive gym website built with a modern no-build stack:

- HTML5 (semantic structure)
- CSS3 (mobile-first, custom properties, responsive grid)
- Vanilla JavaScript (mobile navigation behavior)

## Run locally

Open `index.html` directly in your browser.

## Sections included

- Hero with CTA and stats
- About
- Programs
- Trainers
- Pricing
- Contact form
